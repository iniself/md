1. **命令面板** =grey:: ⌘ ::=
    该面板集合了 “最近文件”、“搜索文件” 和 “搜索命令” 功能。通过按钮 =grey:: ⌘ ::= 或快捷键 [`Cmd + O`]{ =default:: macOS ::= } / [`Ctrl + O`]{ =default:: Windows ::= } 打开

2. **复制**
    * `公众号`、`知乎`点击复制后，粘贴到对应平台
    * `HTML` 点击复制后，粘贴到需要嵌入的地方
    * `MD` 就是复制当前编辑器的内容

3. **编辑**
    * 图片可直接在编辑器中粘贴，或通过`上传图片`菜单。无论哪种方式都需要先在该菜单进行图床配置。=default:: 💡 本地图床仅桌面端软件有 ::=
    * `插入表格`可交互式插入表格

4. **文件**
    * `本地文件夹`、`保存到文件`与本地文件同步有关。默认文章是保存在浏览器缓存中，可以通过`保存到文件`和`本地文件夹`来保存和打开本地文件
    * 各`导入`、`导出`功能如其名
    * `导入/导出项目配置`用于不同系统、设备之间的配置同步

5. **Slash Menu**
    在输入文字的地方（编辑区），按 `/` 时会出现 slash menu。方便插入各种模版语法，比如 `mermaid`、`infographic`、`数学公式`等

6. **弹出菜单**
    编辑区鼠标右键会弹出便捷菜单

7. **文件列表/文章大纲**
    点击可以在 `文件列表`和当前`文章大纲`之间切换

8. **右侧面板**  =grey:: {fa-solid fa-gear} ::=
    该面板主要用于 Docs^red:+^ 各项配置。不容易理解的都有说明

9.  **预览区**
    预览区就是渲染 Markdown 的区域
    * 在该区域点击`图片`可以迁移该图片到其他图床；点击`公式`可以进入交互式修改公式界面；点击 `Infographic` / `Mermaid` 可以转成图片并上传到指定图床
    * 按住 [`Cmd`]{ =default:: macOS ::= } / [`Ctrl`]{ =default:: Windows ::= } 点击`图片` / `公式` / `Infographic` / `Mermaid` 进入预览模式，并可放大和缩小
